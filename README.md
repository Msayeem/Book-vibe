# 📚 Book Vibe

Book Vibe is a frontend web application built with React and Vite that lets users browse books, view detailed information about each title, and manage a personal reading list — all from a clean, responsive interface.

---

## ✨ Key Features

### 1. 📖 Book Browsing & Detail View
Explore a curated collection of books on the home page. Click any book to navigate to its dedicated detail page, powered by React Router for smooth, client-side navigation.

### 2. 📑 Reading List Management
Add books to a personal "Read" or "Wishlist" collection. Your selections are stored locally so you can track what you've read and what you want to read next.

### 3. 📱 Responsive UI
Built with Tailwind CSS and Bootstrap, the app adapts seamlessly across desktop and mobile screen sizes, delivering a consistent experience on any device.

## 🚀 Installation & Local Setup

### Prerequisites
- [Node.js](https://nodejs.org/) (v18 or higher recommended)
- npm (comes with Node.js)

### Steps

1. **Clone the repository**
```bash
   git clone https://github.com/Msayeem/Book-vibe.git
   cd Book-vibe
```

2. **Install dependencies**
```bash
   npm install
```

3. **Start the development server**
```bash
   npm run dev
```

4. **Open in your browser**

Once started, open the local link provided in your terminal (usually http://localhost:5173) to view the app
